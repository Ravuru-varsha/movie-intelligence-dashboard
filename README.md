# movie-intelligence-dashboard
Movie analytics and ML project using Python, Power BI and scikit-learn
## Overview
End-to-end data analytics and ML project analyzing 3229 movies 
from the TMDB 5000 dataset.
## Tools Used
- Python (pandas, matplotlib, seaborn, scikit-learn)
- Power BI (interactive dashboard)
- Google Colab
- Microsoft Excel (data preprocessing & cleaning)
- ## What I Built
- EDA and data cleaning
- Power BI dashboard with 5 interactive charts
- Linear Regression → predict movie revenue (R² = 0.558)
- Random Forest Classifier → predict Hit or Flop (82.51% accuracy)
- K-Means Clustering → grouped movies into 4 segments

## Key Insights
- High budget correlates with revenue but not always with ratings
- 76% of movies are Flops, only 24% are Hits
- Mega Hit movies average $833M revenue vs $163M budget
